SVN Plugin is an interface for connecting to a SVN repository where
the tests are located and it will replicate the content locally.
After you download and activate the plugin a new tab will appear
called SVNPlugin. You can configure the how to connect to the repository.
SVN Plugin has a few parameters to be configured:
    - server complete path
    - user and password to connect to server
    - snapshot folder, where all data is cloned
